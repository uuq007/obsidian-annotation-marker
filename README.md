# Obsidian Annotation Marker

为 Obsidian 笔记添加文本标注、高亮、批注和注音功能。

标注数据存储在独立的标注文件中，不修改原始 Markdown 文件。支持预览模式和编辑模式。

## 功能特性

### 标注类型

- **彩色标注** — 5 种颜色高亮（红、蓝、黄、绿、紫），可自定义颜色
- **批注** — 为标注添加详细笔记，悬浮即可查看
- **注音** — 为选中文本添加注音（ruby 标签），支持多个注音
- **全文标注** — 一键标注文件中所有相同文本
- **跨段标注** — 支持跨越段落和文本块的标注
- **嵌套/重叠标注** — 自动处理标注重叠区域的渲染

### 交互操作

- 选中文字后直接选择颜色添加标注
- 点击标注弹出操作菜单（编辑颜色、批注、注音、删除）
- 鼠标悬浮显示批注内容气泡
- 标注列表面板 — 浮动显示当前文件所有标注，点击可跳转
- 侧边栏标注管理视图 — 按文件浏览全部标注，支持搜索和颜色筛选

### 双模式支持

- **预览模式** — 标注以彩色 `<mark>` 标签渲染
- **编辑模式** — 通过 CodeMirror 6 扩展显示标注高亮和注音

### 文件管理

- 标注数据独立存储，原始文件零侵入
- 原文件修改后自动同步到标注文件（基于 diff 算法）
- 原文件重命名/删除时自动迁移/清理标注数据
- 支持从旧版插件导入标注数据

### 可配置项

- 默认标注颜色
- 5 种标注颜色的自定义（支持十六进制色值）
- 颜色标签名称
- 批注最大长度
- 批注视觉效果（无/粗线/虚线/波浪线/双线）
- 注音字体大小和颜色

## 安装

### 手动安装

1. 从 [Releases](https://github.com/uuq007/obsidian-annotation-marker/releases) 下载最新版本
2. 将 `main.js`、`manifest.json`、`styles.css` 复制到 vault 的 `.obsidian/plugins/obsidian-annotation-marker/` 目录
3. 在 Obsidian 设置中启用插件

### 从源码构建

```bash
git clone https://github.com/uuq007/obsidian-annotation-marker.git
cd obsidian-annotation-marker
npm install
npm run build
```

## 使用方法

1. 点击左侧边栏的标注图标进入标注视图
2. 选中文字后弹出颜色选择菜单
3. 点击已有标注可编辑或删除
4. 通过命令面板可执行：
   - `切换标注视图` — 在普通视图和标注视图间切换
   - `标注管理侧边栏` — 打开标注管理面板
   - `导入旧标注插件数据` — 从旧版插件迁移数据

## 与旧版的区别

v3.0.0 是完全重写的版本，主要变化：

- 标注数据不再使用 JSON 文件存储，改为嵌入到独立的 Markdown 标注文件中
- 支持编辑模式（CodeMirror 6）
- 新增侧边栏标注管理视图
- 新增全文标注、跨段标注、注音功能
- 更好的标注重叠处理
- 提供旧版数据导入命令，可从旧版平滑迁移
