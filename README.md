# Obsidian Annotation Marker

**作者**：uuq007

在不改动原始 Markdown 文件的前提下，为 Markdown 文件添加标注、高亮和批注。

## 功能特点

- ✨ **不修改原文件**：所有标注数据独立存储，不影响原始 Markdown 文件
- 🎨 **多种标注颜色**：支持红色、蓝色、黄色、绿色、紫色、无色等多种标注颜色
- 📝 **添加批注**：为标注添加详细的批注说明
- 🔤 **注音功能**：为文字添加注音（Ruby 标签）
- 🔄 **自动同步**：支持文件重命名、移动时自动迁移标注数据
- 🗑️ **自动清理**：文件删除时自动清除相关标注数据
- 🎯 **智能定位**：采用渐进式搜索策略，即使文档内容变化也能准确定位标注
- 📱 **跨设备兼容**：同时支持桌面端和移动端

## 安装

### 方法一：手动安装

1. 下载最新版本的发布包
2. 将 `main.js`、`manifest.json` 和 `styles.css` 复制到 Obsidian 的插件目录：
   ```
   <Your Vault>/.obsidian/plugins/obsidian-annotation-marker/
   ```
3. 在 Obsidian 的 **设置 → 社区插件** 中启用插件

### 方法二：从 Obsidian 插件市场安装（待发布）

1. 打开 Obsidian 的 **设置 → 社区插件**
2. 搜索 "Obsidian Annotation Marker"
3. 点击安装

## 使用方法

### 开启标注模式

- 点击左侧边栏的🖌️图标
- 或使用命令面板（`Ctrl/Cmd + P`）执行"切换标注模式"命令

### 添加标注

1. 在标注模式下选中要标注的文本
2. 选择标注颜色
3. （可选）添加批注内容
4. （可选）为文字添加注音
5. 点击"保存"

### 查看批注

- 将鼠标悬停在标注文本上，会显示批注内容

### 管理标注

- 在左侧标注列表中查看所有标注
- 点击标注项可以编辑或删除

### 关闭标注模式

- 再次点击左侧边栏的🖌️图标
- 或使用"切换标注模式"命令

## 功能说明

### 标注颜色

| 颜色 | 用途建议 |
|------|---------|
| 🔴 红色 | 重要内容、需要关注的点 |
| 🔵 蓝色 | 一般性标注、补充说明 |
| 🟡 黄色 | 默认颜色、一般标注 |
| 🟢 绿色 | 已解决问题、已完成事项 |
| 🟣 紫色 | 特殊标注、个人笔记 |
| ⚪ 无色 | 仅添加批注，不改变文本外观 |

### 注音功能

- 支持为选中文本的任意部分添加注音
- 可以为一个标注添加多个注音
- 注音以 `<ruby>` 标签形式显示

### 数据存储

- 标注数据以 JSON 格式存储在插件目录中
- 每个 Markdown 文件对应一个独立的标注数据文件
- 数据位置：`<Vault>/.obsidian/plugins/obsidian-annotation-marker/annotations/<文件路径>.json`

### 智能定位机制

插件采用先进的定位策略：

1. **精确匹配**：使用"上文+文本+下文"进行精确匹配
2. **渐进式搜索**：从原位置附近开始，逐步扩大搜索范围
3. **模糊匹配**：当精确匹配失败时，使用相似度算法找到最佳匹配
4. **自适应更新**：找到标注后自动更新上下文信息

即使文档内容发生变化，标注也能准确定位到正确位置。

## 技术特点

### 定位机制

- **排除非正文元素**：自动排除文件名、笔记属性（metadata/frontmatter）、标题栏等
- **渐进式搜索**：从局部到全局，性能更优
- **双重匹配策略**：精确匹配 + 模糊匹配，确保不丢失标注
- **相似度算法**：使用 Levenshtein 距离计算上下文相似度

### 性能优化

- 多次小搜索，命中更快
- 智能搜索范围，避免全文档遍历
- 自动上下文更新，减少重复计算

### 数据管理

- 独立的数据存储，不污染原文件
- 自动迁移标注数据，支持文件重命名
- 自动清理标注数据，支持文件删除
- 缓存机制，提升读取性能

## 开发说明

### 项目结构

```

│   ├── main.ts              # 插件入口

│   ├── dataManager.ts        # 数据管理
│   ├── types.ts              # 类型定义
│   └── utils/
```

3. 创建 GitHub Release
4. 上传 `main.js`、`manifest.json` 和 `styles.css` 到 Release
5. 提交到 [Obsidian 插件目录](https://github.com/obsidianmd/obsidian-releases)

## 使用限制

- 标注模式仅在预览模式下可用
- 切换到编辑模式会自动退出标注模式
- 标注数据存储在本地，不会随文件同步到其他设备（需要额外同步配置）
